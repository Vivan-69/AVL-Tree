# AVL Tree Visualization

An interactive, web-based visualization tool for an AVL Tree. This project provides a step-by-step graphical representation of AVL tree operations, including insertion, deletion, searching, and traversal (print), replicating the classic visualization by David Galles at the University of San Francisco.

## Features

*   **Interactive Operations**:
    *   **Insert**: Add a numerical value to the AVL Tree, accompanied by a step-by-step visual animation demonstrating the balancing process (rotations).
    *   **Delete**: Remove a node from the tree and observe the rebalancing animations.
    *   **Find**: Search for a specific value in the tree, with the search path highlighted.
    *   **Print**: Traverse and output the tree contents.
*   **Animation Controls**:
    *   Adjust the animation speed using the provided slider.
    *   Step Forward, Step Back, Skip Forward, Skip Back, and Pause the current animation sequence to follow the algorithm step-by-step.
*   **Customizable Canvas**: Ability to resize the drawing area dynamically.

## Technologies Used

*   **HTML5**: For structuring the interface and providing the drawing canvas.
*   **CSS3**: For styling the layout and interactive controls.
*   **JavaScript**: Handles tree logic, algorithms, animations, and DOM manipulations.

## Setup and Usage

1.  Clone this repository to your local machine.
2.  Open `index.html` in any modern web browser.
3.  Use the input fields and buttons at the top of the page to start interacting with the tree.
